# 池田英会話スクール生徒管理システム

池田英会話スクールの生徒情報を管理するためのシステム
システムの機能は２つのモードの別れています。
    ー管理者
    ー生徒

## 機能

### ログイン
    システム起動時にIDとパスワードを問われますシステムはIDとパスワードを確認し、正しいものであればログインを許します。

### 生徒機能
    生徒がログインした時に使用できる機能
#### レッスンの予約
日時を指定してレッスンの予約ができます。
#### レッスンの予約状況の確認
予約済みのレッスンの日時を確認できます。
#### ポイントの購入
クレジットカードの番号を入力することで、ポイントを購入できます。
#### パスワードの変更
パスワードを変更できます。

### 管理者画面
    先生や受付がログインする時の画面
    
#### 生徒入会
生徒の個人情報を登録します。
#### 各種登録情報の表示
生徒・教師・予約の現在の状態を確認できます。
#### 生徒情報の変更
生徒の情報を書き換えられます。（データを書き換える）
#### 教師の追加
新しく教師を登録できます。
